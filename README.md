# actuarial-life-model-engine
=======
## Assumptions Module

This module defines the core assumptions used in the model:

- Mortality (`qx`): Probability of death by age, using a synthetic curve
- Survival (`px`): Derived from mortality
- Discounting: Based on a flat interest rate

This structure mirrors how assumptions are handled in traditional actuarial models, where mortality tables and yield curves are central inputs into projections.
